# research

Automated research assistant that searches [libgen.vg](https://libgen.vg), [arXiv](https://arxiv.org), and [IACR ePrint](https://eprint.iacr.org) for papers/books on a given topic, downloads PDFs, extracts excerpts (TOC + first N pages), and synthesizes per-source summaries into `idea.md` and comparative `research.md`.

## Features

- Multi-source search: libgen.vg (via Tor), arXiv (direct), IACR ePrint (direct)
- Results interleaved and deduplicated across sources
- Download PDFs with automatic mirror resolution and retry
- Extract TOC and first pages from PDFs
- Generate per-source summaries, cross-source synthesis (`idea.md`), and comparative analysis (`research.md`)
- Claude Code `/research` slash command for end-to-end workflow

## Requirements

- Python 3.12+
- Tor proxy running on `127.0.0.1:9050` (for libgen access)

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r scripts/requirements.txt
```

Start the Tor proxy:

```bash
/opt/homebrew/opt/tor/bin/tor --quiet &
```

## Usage

### CLI

```bash
source .venv/bin/activate
HTTPS_PROXY=socks5h://127.0.0.1:9050 \
  python scripts/libgen_fetch.py "zero knowledge proof" --top 10 --pages 30
```

Output is written to `topics/YYYY-MM-DD_<slug>/` containing:
- `sources/` — downloaded PDFs and extracted excerpts
- `summaries/` — per-source summaries (generated by `/research`)
- `metadata.json` — run metadata and per-source status
- `idea.md` — cross-source synthesis (generated by `/research`)
- `research.md` — comparative analysis (generated by `/research`)

### Claude Code

```
/research zero knowledge proof
```

This runs the full pipeline: fetch from all sources, summarize each paper, synthesize `idea.md`, and generate comparative `research.md`.

## Project Structure

```
scripts/
  libgen_fetch.py       # CLI entry point (multi-source)
  libgen/
    search.py           # libgen.vg search parser
    arxiv.py            # arXiv API search
    iacr.py             # IACR ePrint search parser
    download.py         # mirror resolver and file downloader
    extract.py          # PDF TOC and excerpt extractor
    metadata.py         # run metadata and JSON writer
    slug.py             # topic slug utilities
tests/
  test_search_parser.py # search HTML parsing tests
  test_slug.py          # slug generation tests
topics/                 # research output (per-topic directories)
```
